## The Coral Ecosystem

Coral is a collection of independent, modular interfaces for [Jellyfin](https://jellyfin.org/) — your self-hosted media system. Each module is purpose-built for a specific media type or experience.

All modules are built with:
- **TanStack Start** for modern server-side rendering
- **Tailwind CSS v4** for styling
- **Jellyfin API** as the backend
- **TypeScript** for type safety
